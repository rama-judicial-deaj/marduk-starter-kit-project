## Installation

1. Clone the repository:
   ```bash
   git clone https:// 
   ```

2. Navigate to the project directory:
   ```bash
   cd  
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to:
   ```
   http://localhost:3002
   ```
